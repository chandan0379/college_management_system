# 🎓 College Management System

A full-stack College Management System developed using **Python Flask**, **SQLAlchemy**, **HTML**, **CSS**, **Bootstrap**, and **SQLite**.

This project helps manage students, teachers, librarians, books, and academic records through separate role-based portals.

## 🚀 Live Demo

https://college-management-system-d7h7.onrender.com

---

## ✨ Features

### 👨‍🎓 Student Module

* Student Login
* Student Dashboard
* View Profile
* Upload Profile Photo
* Upload Signature
* View Attendance
* View Marks
* View Library Books
* View Issued Books

### 👨‍🏫 Teacher Module

* Teacher Login
* Teacher Dashboard
* Manage Student Records

### 📚 Library Module

* Librarian Login
* Add Books
* Delete Books
* Issue Books
* Manage Library Inventory
* Track Issued Books

### 🛠 Admin Module

* Admin Login
* Register Students
* Register Teachers
* Manage Student Data
* Manage Teacher Data
